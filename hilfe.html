ChordPro Studio Hilfe

Willkommen bei ChordPro Studio, einem professionellen Tool zum Erstellen, Gestalten und Exportieren von Songblättern im ChordPro-Format.

1. Der Editor

Der Editor verwendet das standardmäßige ChordPro-Format. Akkorde werden in eckigen Klammern [ ] innerhalb des Songtexts platziert.

{title: Hallelujah}
{subtitle: Leonard Cohen}

[C]I heard there was a [Am]secret chord
That [C]David played and it [Am]pleased the Lord
But [F]you don't really [G]care for music, [C]do you? [G]

Häufige Direktiven sind:

  • {title: Name} - Songtitel
  • {subtitle: Künstler} - Name des Künstlers
  • {comment: Text} - Allgemeine Kommentare oder Anweisungen
  • {start_of_chorus} / {soc} - Markiert den Beginn eines Refrains
  • {end_of_chorus} / {eoc} - Markiert das Ende eines Refrains

2. Stil & Layout

Verwenden Sie die obere Navigationsleiste, um das Aussehen Ihres Songblatts anzupassen:

  • Zeilenabstand: Passen Sie den vertikalen Abstand zwischen den Zeilen an.
  • Schriftarten: Wählen Sie verschiedene Schriftarten für Titel, Künstler, Songtext, Akkorde und Kommentare.
  • Größen & Farben: Ändern Sie die Schriftgröße und Farbe für jedes Element einzeln.
  • Refrain-Styling: Schalten Sie Einrückungen und vertikale Linien für Refrain-Blöcke ein oder aus.

3. Strumming-Pattern-Editor

Dieses Tool in der linken Seitenleiste hilft Ihnen, Rhythmen zu visualisieren:

  • Wählen Sie eine Taktart (4/4, 3/4 oder 6/8).
  • Klicken Sie auf die Slots, um Abschläge (↓), Aufschläge (↑) oder leere Beats umzuschalten.
  • Passen Sie BPM und Skalierung an, um die Größe und Geschwindigkeits-Metadaten zu ändern.
  • Klicken Sie auf "Insert Pattern", um die Direktive zu Ihrem Song hinzuzufügen.

4. Akkord-Diagramm-Editor

Erstellen Sie benutzerdefinierte Akkorddiagramme, wenn ein Akkord nicht in der integrierten Bibliothek enthalten ist:

  • Geben Sie den Akkordnamen ein.
  • Klicken Sie auf das Griffbrett-Raster, um Finger (Punkte) zu platzieren.
  • Geben Sie bei Bedarf Barre-Bünde an.
  • Klicken Sie auf "Define", um den benutzerdefinierten Akkord zu Ihrem Song hinzuzufügen.

5. PDF-Export

Wenn Sie bereit sind, klicken Sie auf die Schaltfläche "PDF Export" im Header. Sie können zwischen einem einspaltigen oder zweispaltigen Layout wählen, bevor Sie Ihre Datei speichern.

6. Tastaturkürzel

Fügen Sie beim Tippen schnell gängige Akkorde ein. Wir empfehlen die Verwendung von Alt + [Buchstabe], um Konflikte mit Standard-Browser-Kürzeln (wie Strg+C zum Kopieren) zu vermeiden:

  • Alt + A: [A]
  • Alt + B: [B]
  • Alt + C: [C]
  • Alt + D: [D]
  • Alt + E: [E]
  • Alt + F: [F]
  • Alt + G: [G]

Der Cursor wird innerhalb der Klammern nach dem Buchstaben platziert, sodass Sie sofort Modifikatoren wie 'm' oder '7' hinzufügen können.
